주어진 문장
For many people though, we spend far too much time on the other end of the scale, doing what is referred to as shallow work.
Deep work is when you perform an activity with deep concentration that pushes your cognitive capabilities, and by doing so create new value and improve your skills.
( 1 ) These efforts are hard to replicate; and to be able to do deep work, we need a distraction-free environment.
( 2 ) This style of working is not cognitively demanding but is rather an execution of logical-type tasks that rarely create new value (although they may still be important). ← 정답 위치
( 3 ) We often perform these tasks while we are distracted.
( 4 ) Shallow work has its place, both because routine-type tasks need to be done and because we cannot spend our entire time doing deep work.
( 5 ) The problem with too much shallow work is that we feel busy, maybe even stressed, but we rarely see any output reflecting our experienced effort.
