FULL PASSAGE
The standard school mathematics curriculum doesn't stress enough that mathematics is all about abstraction.
Mathematicians study specific objects but aim to cluster them together and study them all at the same time.
① This saves huge amounts of time and effort — properties need only be proved once for the whole class.
② This guarantees that every object within a class has a particular property, even undefined objects.
③ Often, mathematicians prove an object belongs to a class already known to have the desired property.
④ Mathematical calculation ability serves as a key indicator of academic development. ← 흐름 무관
⑤ We reason similarly in everyday life: all fruits have seeds → apple is a fruit → apple must have seeds.

C. 중요한 대상이 이미 알려진 집합에 속함을 증명하면 그 성질도 자동으로 증명된다.

💬 모든 과일에 씨가 있고 사과가 과일이면 사과에도 씨가 있다 — 이게 수학적 추상화의 일상 버전이야.
